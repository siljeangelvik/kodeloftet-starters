Terminal

* H O M E B R E W *

Check if Homebrew is installed: command -v brew

Check if Homebrew is installed the path (/usr/local/bin/brew): which brew

Display the current state of Homebrew’s analytics: brew analytics [state]

Check Homebrew version: brew --version

Upgrade everything with: brew upgrade

Or upgrade a specific formula with: brew upgrade [formula]

Check all the packages installed with Homebrew: brew list

List which of your installed packages (kegs) are outdated with: brew outdated

Update all package definitions (formulae) and Homebrew itself: brew update

Update all the outdated formulae: brew upgrade

Uninstall package: brew uninstall < package_name >

* N P M *

List all the globally installed npm packages: npm list -g --depth=0

Uninstall package: npm uninstall -g < package_name >

Terminal Homebrew Warnings / Errors

WARNING: Homebrew's "sbin" was not found in your PATH but you have installed formulae that put executables in /usr/local/sbin.

Consider setting your PATH for example like so:

echo 'export PATH="/usr/local/sbin:$PATH"' >> ~/.zshrc

REASON

Apple macOS has moved from Bash shell to Zsh (Z Shell).

SOLUTION:

Run the command below in your terminal:

echo 'export PATH="/usr/local/sbin:$PATH"' >> ~/.zshrc

Now quit the terminal by pressing command+q, re-open the terminal and run brew doctor

PACKAGES

rtmpdump: This package is useful for downloading RTMP streams. You might need it if you frequently download or work with content from RTMP servers.

slides: This package is used for creating and displaying presentation slides. It could be handy for creating slide decks for presentations or lectures.

tidy-html15: Tidy is a tool to clean up and pretty-print HTML files. You might use it to ensure your HTML files adhere to standards and are well-formatted.

m4: M4 is a macro processor typically used in software development or system administration tasks for text processing. It's often used in conjunction with other tools for template processing or code generation.

openssl@3 and openssl@1.1: These are different versions of OpenSSL, a cryptographic library commonly used in secure communication protocols such as HTTPS. You might have both versions installed for compatibility reasons, or because certain applications require specific versions of OpenSSL.
